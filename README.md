# Cache Store

A key-value store for generating radom data for your keys

## Getting Started

### Prerequisites

- Node v8.9.4
- Mongo v4.0.2
- Yarn 1.10.1

### Installing

1. Clone project  `git clone https://github.com/ydaniju/cache-store`

2. Run `cd cache-store`

3. Run `yarn`

4. Start server in development using `yarn start:dev`

## Running the tests

### Unit Tests

Run `yarn test`

### Coding style tests

Run `yarn lint`
